CineReviews

Simulating an online review system for movies and series.

Java Programming | Object-Oriented Design | Data Management

My Role

As part of the Object-Oriented Programming course during the second year of university, I was tasked with developing a Java application called CineReviews. This project simulates an online review system where users can register as critics, audience members, or admins, and review movies and series. I was responsible for the core logic, object-oriented design, and command-line interface.

Challenges & Strategy

The main challenge was implementing the complex rules of user roles, show management, and review aggregation. Additionally, managing the dynamic nature of user interactions, show uploads, and artist information required careful planning.

  • Solution: I used object-oriented programming principles to create classes for users, shows, artists, and reviews, ensuring modularity and reusability.
  • Solution: I implemented a command-line interface to handle user input and output, following the exact specifications provided in the project brief.
  • Solution: I conducted thorough testing to ensure that all edge cases, such as duplicate reviews and show uploads, were handled correctly.

Results & Impact

  • Functional Application: The final product was a fully functional Java program that could simulate an online review system, complete with all the rules and mechanics described in the project brief.
  • Positive Feedback: The project was well-received by the course instructors, who praised the clarity of the code and the robustness of the simulation logic.
  • Learning Experience: This project significantly improved my understanding of object-oriented design, Java programming, and problem-solving in a data management context.
