Phong Lighting

Visualizing Gouraud and Phong shading techniques.

WebGL | GLSL | Phong Illumination | Gouraud Shading | Phong Shading

My Role

As part of the Computer Graphics and Interaction course, I was tasked with implementing the Phong Illumination Model in WebGL, supporting both Gouraud shading (per-vertex lighting) and Phong shading (per-fragment lighting). This project involved creating an interactive interface using the dat.GUI library, managing light sources, and rendering objects with dynamic material properties. I was responsible for the lighting calculations, shader implementation, and user interaction.

Challenges & Strategy

The main challenge was implementing the Phong Illumination Model in both Gouraud and Phong shading techniques while ensuring real-time updates to the scene based on user input. Additionally, managing multiple light sources with different reference frames (world, object, and camera) required careful handling of transformations and coordinate systems.

  • Solution: I implemented the Phong Illumination Model in GLSL, with separate shaders for Gouraud and Phong shading.
  • Solution: I created an interactive interface using dat.GUI to allow users to manipulate light sources, material properties, and camera settings.
  • Solution: I optimized the rendering pipeline to handle dynamic updates to light positions, material properties, and shading techniques.

Results & Impact

  • Functional Application: The final product was a fully functional WebGL application that allowed users to visualize and interact with objects using Gouraud and Phong shading techniques.
  • Positive Feedback: The project was well-received by the course instructors, who praised the accuracy of the lighting calculations and the flexibility of the interface.
  • Learning Experience: This project significantly improved my understanding of lighting models, shader programming, and interactive 3D graphics.
