Modeling a Fire Truck.

WebGL | GLSL | Interactive Graphics | Hierarchical Modeling

My Role

As part of the Computer Graphics and Interaction course, I was tasked with developing a WebGL application to model and visualize a fire truck using hierarchical transformations. This project involved creating a hierarchical scene graph, implementing interactive controls for manipulating the fire truck's components, and rendering the scene using GLSL shaders. I was responsible for the hierarchical modeling, WebGL rendering pipeline, and user interaction.

Challenges & Strategy

The main challenge was implementing the hierarchical transformations for the fire truck's components, such as the extendable ladder, rotating platform, and wheels. Additionally, creating an interactive interface for manipulating these components required careful handling of user input and real-time updates to the scene.

  • Solution: I implemented a hierarchical scene graph in JavaScript to manage the transformations of the fire truck's components.
  • Solution: I created an interactive interface using JavaScript to allow users to manipulate the fire truck's components using keyboard commands.
  • Solution: I optimized the rendering pipeline to handle complex hierarchical transformations and ensure smooth animations.

Results & Impact

  • Functional Application: The final product was a fully functional WebGL application that allowed users to interactively manipulate and visualize a fire truck with hierarchical transformations.
  • Positive Feedback: The project was well-received by the course instructors, who praised the accuracy of the hierarchical modeling and the smoothness of the rendering.
  • Learning Experience: This project significantly improved my understanding of hierarchical modeling, WebGL, and interactive application development.
