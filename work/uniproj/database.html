Oracle Database | APEX Interface | Data Management

My Role

As part of the Databases course during the second year of university, I was tasked with developing a complete database system using Oracle® 18c and Oracle® Application Express (APEX) for the interface. This project involved designing a database schema, implementing it, and creating a user-friendly interface to interact with the data. I was responsible for the database design, schema implementation, and APEX interface development.

Challenges & Strategy

The main challenge was designing a robust database schema that could handle complex relationships and ensure data integrity. Additionally, creating an intuitive and functional APEX interface required careful planning and iterative development.

  • Solution: I used Oracle® 18c to implement the database schema, ensuring proper normalization and integrity constraints.
  • Solution: I developed an APEX interface to allow users to interact with the database, including forms, reports, and navigation features.
  • Solution: I conducted thorough testing to ensure that all functionalities, such as data insertion, updates, and complex queries, worked as expected.

Results & Impact

  • Functional Database System: The final product was a fully functional database system with a user-friendly interface, complete with all the features described in the project brief.
  • Positive Feedback: The project was well-received by the course instructors, who praised the clarity of the database design and the robustness of the APEX interface.
  • Learning Experience: This project significantly improved my understanding of database design, SQL, and web-based interface development using APEX.
