B-Spline Curves

Visualizing and interacting with cubic B-Spline curves.

WebGL | GLSL | Interactive Graphics

My Role

As part of the Computer Graphics and Interaction course, I was tasked with developing a WebGL application to visualize and interact with cubic B-Spline curves. This project involved implementing the mathematical foundations of B-Splines, creating an interactive interface for curve editing, and rendering the curves using GLSL shaders. I was responsible for the mathematical implementation, WebGL rendering pipeline, and user interaction.

Challenges & Strategy

The main challenge was implementing the mathematical model of B-Spline curves and ensuring smooth rendering using WebGL and GLSL. Additionally, creating an interactive interface for curve editing required careful handling of user input and real-time updates to the curve visualization.

  • Solution: I implemented the B-Spline blending functions in GLSL to compute curve points dynamically on the GPU.
  • Solution: I created an interactive interface using JavaScript to allow users to add and manipulate control points.
  • Solution: I optimized the rendering pipeline to handle multiple curves and ensure smooth animations.

Results & Impact

  • Functional Application: The final product was a fully functional WebGL application that allowed users to create, edit, and visualize B-Spline curves interactively.
  • Positive Feedback: The project was well-received by the course instructors, who praised the accuracy of the mathematical implementation and the smoothness of the rendering.
  • Learning Experience: This project significantly improved my understanding of computer graphics, WebGL, and interactive application development.
